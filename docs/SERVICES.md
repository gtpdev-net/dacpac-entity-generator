# Services Reference

## DACPAC Services

### DacpacExtractorService

Extracts `model.xml` from DACPAC files (which are standard ZIP archives).

| Method | Description |
|---|---|
| `ExtractModelXml(inputDirectory, server, database)` | Extracts from `{inputDir}/dacpacs/{server}_{database}.dacpac` |
| `ExtractModelXmlFromPath(dacpacPath, server, database)` | Extracts from an explicit file path |
| `ExtractModelXmlFromStream(stream, server, database)` | Extracts from an in-memory stream (used by web upload) |
| `DacpacExists(inputDirectory, server, database)` | Checks if the expected DACPAC file exists |

### ModelXmlParserService

Parses the DACPAC `model.xml` (DAC Serialization format) using `System.Xml.Linq`.

| Method | Description |
|---|---|
| `PrepareDocument(modelXml, server, database)` | Parses and validates the XML; returns `XDocument` or null |
| `ParseTable(doc, server, database, schema, tableName, requiredColumns)` | Parses a single table with filtered columns |
| `ParseAllTables(doc, server, database)` | Parses all tables with all columns |
| `ParseViews(doc, server, database)` | Parses all views with columns and SQL bodies |
| `ParseStoredProcedures(doc, server, database)` | Parses stored procedures with parameters |
| `ParseUserDefinedFunctions(doc, server, database)` | Parses user-defined functions |
| `ParseTriggers(doc, server, database)` | Parses table-level triggers |
| `GenerateDiscoveryReport(doc, server, database)` | Generates an `ElementDiscoveryReport` |

Uses DACPAC XML namespace: `http://schemas.microsoft.com/sqlserver/dac/Serialization/2012/02`

### PrimaryKeyEnricher

Validates that tables have valid primary key information after parsing.

| Method | Description |
|---|---|
| `EnrichTableWithPrimaryKeys(table)` | Validates PK columns exist; logs auto-added PKs |

## Import Services

### DacpacSchemaImportService

Imports full DACPAC schema into the DataManager database.

| Method | Description |
|---|---|
| `ImportDacpacFolderAsync(folderPath)` | Imports all `.dacpac` files from a directory |
| `ImportSingleDacpacAsync(stream, serverName, databaseName)` | Imports a single DACPAC from a stream |

Uses hash-based deduplication: if the `model.xml` SHA-256 hash matches the stored `LastImportedModelHash`, the import is skipped.

Import strategy:
- **Server / SourceDatabase** — Upsert (create if not found)
- **SourceTable / SourceColumn** — Upsert (preserve catalogue metadata)
- **Schema objects** (indexes, FKs, constraints, views, procs, functions, triggers) — Delete-and-replace

### DataManagerImportService

Imports catalogue data (server/database/table/column metadata) from preview rows.

| Method | Description |
|---|---|
| `ImportAsync(preview, strategy, dryRun)` | Imports from `ImportPreviewRow` list |

Conflict strategies:
- `SkipExisting` — Skip columns that already exist
- `AddNewOnly` — Only add new columns
- `FullSync` — Full synchronization with deletes

### ExcelImportService

Reads Excel files and produces `ImportPreviewRow` lists for the import pipeline.

### MigrationConfigLoadService

Loads migration configuration from external sources into the `MigrationConfigs` table.

## Data Access

### IDataManagerRepository

The primary data access interface with 40+ methods covering all CRUD operations, bulk operations, and queries. See [DATA-MODEL.md](DATA-MODEL.md) for entity details.

Key method groups:

| Group | Methods |
|---|---|
| **Servers** | Get, Add, Update, Delete, GetByRole, NameExists |
| **Server Connections** | Get, Save |
| **Target Databases** | Get, Add, Update, Delete, NameExists |
| **Source Databases** | GetInScope, GetById, Add, Update, Delete |
| **Source Tables** | GetInScope, GetById, Add, Update, Delete |
| **Source Columns** | Get (with filters), GetById, Add, Update, Delete, BulkUpdate |
| **Schema Objects** | Views, StoredProcedures, Functions, Triggers, Indexes, ForeignKeys, CheckConstraints, UniqueConstraints |
| **Schema Import** | UpdateDatabaseHash, DeleteSchemaForDatabase, BulkInsert* |
| **Migration Config** | Get, GetById, Add, Update, Delete, BulkUpsert |
| **Activity Log** | LogCopySuccess, LogCopyFailure |
| **Summary** | GetDataManagerSummary |
| **Uniqueness** | ServerNameExists, DatabaseNameExists, TableNameExists, ColumnNameExists |

### DataManagerDbContext

EF Core context with 20+ DbSets. Uses `OnModelCreating` for Fluent API configuration of all entities including:
- Table names, primary keys, and unique indexes
- Foreign key relationships with cascade behavior
- Default values and SQL default expressions
- Enum-to-string conversions
- Maximum lengths and required properties

### DataManagerDbSchemaDataSource

Implements `ISchemaDataSource` to read schema from the DataManagerDb for generation. Filters columns by:
- `IsActive = true`
- `IsSelectedForLoad = true`
- `PersistenceType ≠ 'D'`

Supports single-database or all-databases mode via the `DatabaseId` property (negative value = all databases).

## Web Services

### BlazorGenerationLogger

Blazor-compatible implementation of `IGenerationLogger`. Fires an `OnLog` event that Blazor components subscribe to for real-time streaming of generation progress.

```csharp
public event Action<string, string>? OnLog;  // (level, message)
// Levels: "progress", "warning", "error", "info"
```

### NotificationService

Provides toast notification management for the Blazor UI.

### BreadcrumbItem

Simple model for breadcrumb navigation in the UI.

## Utility Classes

### SqlTypeMapper

Maps SQL Server data types to C# types with proper nullability handling.

### NameConverter

Converts SQL identifiers to valid C# PascalCase names with keyword escaping.

### EntityGenerationHelpers

Shared helpers for default value parsing and backing-field name generation:
- `GenerateBackingFieldName(propertyName)` — e.g., `IsActive` → `_isActive`
- `DetermineDefaultBoolValue(defaultValue)` — Parses SQL bool defaults
- `DetermineDefaultIntValue(defaultValue)` — Parses SQL integer defaults
- `TranslateSqlDefaultForSQLite(sqlDefaultValue)` — Translates SQL Server defaults to SQLite equivalents

### HashHelper

Computes SHA-256 hex digests for DACPAC change detection.
